For this part of the homework, I first found if there was a red card in the first 15 minutes of the games. It is called as redBef15 in the booking data. Also using the matches data and extra time scores, I calculated the result of the game at 90th minute, called it as result90. result90 is then compared with result calculated before, in order to be able to say if draw outcome is affected by extra minutes.

DTbookingData[,"redBef15":=ifelse(as.numeric(time)<15&card == "red card" , 1 , 0)]
DTpremierMatches[,"result90" := ifelse(goal_1-match_hometeam_extra_score>goal_2-match_awayteam_extra_score,1,ifelse(goal_2-match_awayteam_extra_score>goal_1-match_hometeam_extra_score,2,0))]

As seen below, variable extraChange becomes 1 if draw outcome changed at the extra minutes or there is a draw outcome due to extra minutes.

DTpremierMatches[,"extraChange" := (result == 0 & result90 != 0) | (result != 0 & result90 == 0)]

setkey(DTbookingData,match_id)
